# Winter OS

A Windows 11-styled personal landing page.

## Features
- Windows 11 aesthetic with boot screen
- Taskbar with live clock
- Desktop icons
- Start menu with avatar
- Discord status via Lanyard
- Light/dark theme toggle

## Pages
- `/` - Main landing page
- `/nohello` - No-hello protocol page
- `/now` - Live clock
- `/fakeisloading` - Fake loading page

## Run locally
```bash
npm install
npm start
```

## Tech
- Express.js
- Vanilla HTML/CSS/JS
- Lanyard API for Discord status
